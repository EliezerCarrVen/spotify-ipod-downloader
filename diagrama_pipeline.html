Importador Gremlin API
Cydia Gremlin / JODebox
  • Vincula archivos multimedia al indexador nativo de iOS.
  • Evita la necesidad de usar iTunes/CopyTrans.
  • Procesa letras integradas directamente en la app de música.
  • Agrupa y actualiza vistas de listas sin reiniciar el dispositivo.
  • Ligero y estable: idóneo para los 512 MB de RAM de este chip A5.
🔄
Sincronización WiFi Incrementable
Delta Sync + Paramiko
  • Verifica la base de datos SQLite antes de cada sincronización.
  • Si la canción ya está en el iPod, no realiza transferencia de red.
  • Permite la desconexión durante la transferencia sin corromper el iPod.
  • Genera registros (logs) detallados si se pierde la conexión.
  • Bajo consumo de CPU en el iPod mediante transacciones rápidas.
