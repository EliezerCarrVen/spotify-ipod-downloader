Una app de escritorio para sincronizar tu música al iPod.

Esta propuesta reemplaza el menú de consola por una experiencia visual: navegación lateral, revisión de biblioteca, cola de descargas, reproductor integrado y preparación para empaquetar el proyecto como ejecutable de Windows.

Objetivo Build portable .exe UI lista para evolucionar a app desktop con Python + WebView/Tauri/Electron o PyInstaller.
